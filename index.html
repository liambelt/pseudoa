Pseudo Elementos

❝active❞

Este enlace cambiará a color blanco mientras hace clic en él.

Representa un elemento (como un botón) que el usuario está activando.

❝any-link❞

Enlace externo

La pseudo-clase :any-link de CSS representa a un elemento que actúa como el ancla origen de un hipervínculo independientemente de si ha sido visitado.

❝first-letter❞

Eilish no solo canta, sino que toca el piano y además compone sus propias canciones, desde la letra, hasta los. arreglos musicales

El pseudoelemento CSS aplica estilos a la primera letra de la primera línea de un elemento de nivel de bloque , pero solo cuando no está precedido por otro contenido (como imágenes).

❝first-line❞

Al igual que muchos otros iconos de la música (desde Michael Jackson hasta Britney Spears), Billie comenzó a explorar su amor por la música cuando era apenas una niña. A los 8 años se unió al coro de niños de Los Ángeles y practicaba canto todos los días, incluso escribió su primera canción a los 11 años.

El pseudoelemento CSS aplica estilos a la primera línea de un elemento a nivel de bloque .

❝checked❞

Representa cualquier radio, checkbox u opción en un elemento que está marcado o conmutado a un estado

❝default❞

Este pseudoelemento selecciona elementos de forma que están predeterminados en un grupo de elementos relacionados.

❝defined❞

Everybodys everything

Representa cualquier elemento que ha sido definido. Incluye cualquier elemento estándar del navegador.

❝dir ()❞

test2

El pseudo-elemento coincide con los elementos en función de la direccionalidad del texto.

❝disabled❞

Representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado (por ejemplo ser selecionado.

❝empty❞

color.

La pseudo-clase representa cualquier elemento que no tenga hijos. Los hijos pueden ser nodos de un elemento o texto.

❝enabled❞



La pseudo-clase -enabled CSS representa cualquier elemento habilitado.

❝first❞

Primera página.

Segunda página.

La pseudo-clase -first de CSS, utilizada con la regla-at @page, representa la primera página de un documento impreso.

❝first-child❞

¡Este texto está seleccionado!

Este texto no está seleccionado.

La pseudo-clase :first-child de CSS representa el primer elemento entre un grupo de elementos hermanos.

❝first-of-type❞

Encabezado

Párrafo 1

Párrafo 2

La pseudo-clase -first-of-type de CSS representa el primer elemento de su tipo entre un grupo de elementos hermanos.

❝focus❞



La pseudo-clase :focus CSS representa un elemento que ha recibido el foco. Generalmente se activa cuando el usuario hace clic, toca un elemento o lo selecciona con la tecla "Tab".

❝focus-visible❞


La pseudo-clase -focus-visible se aplica mientras un elemento coincide con la pseudo-clase -focus y el UA (Agente de Usuario) determina mediante heurística que el foco debe hacerse evidente en el elemento.

❝focus-within❞


La pseudo-clase representa un elemento que ha recibido el foco o que contiene un elemento que ha recibido el foco.

❝fullscreen❞

Demostración :fullscreen

Este texto se pondrá grande y rojo cuando el navegador esté en modo de pantalla completa.

 

La pseudo-clase -fullscreen de CSS representa un elemento que se muestra cuando el navegador está en modo de pantalla completa.

❝hover❞

enlace

La pseudo-clase :hover de CSS coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa.

❝indeterminate❞

Cuyo estado sea indeterminado.

❝in-range❞

    Los valores entre 1 y 10 son válidos.

La pseudo-clase -in-range de CSS representa un elemento cuyo valor actual se encuentra dentro de los límites de rango especificados por los atributos min y max.

❝invalid❞

La pseudo-clase :invalid de CSS representa cualquier elemento ... cuyos contenidos no se puedan validar

❝lang❞

Gdy widzimy znak z narysowaną czaszką i napisem DANGER to lepiej nie wchodzić do środka.

La pseudo-clase es utilizada para modificar elementos en función del idioma en el que se determina que están.
